# いまはこれだけ

iPhone向けPWA試作版です。

## 入っている機能
- 白×ピンクのiPhone風UI
- 体力に応じて今日のタスク上限を1〜3個に制限
- 「次の1個だけ」表示
- やらなくていい項目への避難
- 夜職の枠入力（日付・分数・オプション・手取り・メモ）
- 月間目標金額と進捗
- 毎月の支払いメモ
- 端末内自動保存
- JSONバックアップ／復元
- PWA用manifest・service worker
- 簡易リマインド

## iPhoneのホーム画面に置く
このフォルダをHTTPSで公開し、SafariでURLを開いて「共有」→「ホーム画面に追加」。

## 制限
- 自動クラウド同期にはFirebase/Supabase等のバックエンドが必要です。
- アプリを閉じた後も確実に届くプッシュ通知には、公開URL・Push購読・通知サーバーが必要です。
